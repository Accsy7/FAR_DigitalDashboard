=============================================
  农村集体经济组织三资公示数字大屏 — 环境依赖
=============================================

【运行环境】

  1. Python 3.6+（仅用于启动本地 HTTP 服务器）
     - 使用 Python 标准库 http.server，无需安装任何第三方包
     - 验证方式：命令行输入 python --version

  2. Microsoft Edge 浏览器（或 Chrome）
     - 推荐版本：Edge 90+
     - 分辨率：1920 × 1080

【Python 标准库依赖（无需额外安装）】

  - http.server      — 本地 Web 服务器
  - urllib.request    — 资源下载（仅首次部署时使用）
  - json             — JSON 数据处理
  - os / sys / math  — 文件操作与计算
  - time / random    — 下载限速控制

【前端依赖库（离线，已内置于 lib/ 目录）】

  - Leaflet.js  v1.9.4    — 开源地图引擎
    文件：lib/leaflet.js, lib/leaflet.css
    来源：https://unpkg.com/leaflet@1.9.4/

  - ECharts     v5.5.0    — Apache 图表库
    文件：lib/echarts.min.js
    来源：https://cdn.jsdelivr.net/npm/echarts@5.5.0/

  - Leaflet 图标资源
    文件：lib/images/marker-icon.png
          lib/images/marker-icon-2x.png
          lib/images/marker-shadow.png

【地图瓦片数据（离线，存储于 assets/tiles/ 目录）】

  - 来源：CartoDB Voyager No Labels
    URL模板：https://basemaps.cartocdn.com/rastertiles/voyager_nolabels/{z}/{x}/{y}.png
  - 区域：巴马瑶族自治县及周边
    经度范围：106.30° ~ 108.10°
    纬度范围：23.40° ~ 24.80°
  - 缩放级别：10, 11, 12, 13
  - 瓦片总数：约 2050 张
  - 占用空间：约 30~50 MB
  - 许可协议：© OpenStreetMap contributors, © CARTO

【GeoJSON 边界数据（离线，存储于 data/ 目录）】

  - 来源：阿里云 DataV GeoAtlas
    API：https://geo.datav.aliyun.com/areas_v3/bound/
  - 文件：
    data/guangxi.json   — 广西壮族自治区（区划代码 450000）
    data/hechi.json     — 河池市（区划代码 451200）
    data/bama.json      — 巴马瑶族自治县（从河池市数据中提取）

【第三方包依赖】

  无。本项目完全基于 Python 标准库和离线前端资源，
  不需要安装任何 pip 包（如 Flask、Django 等）。

【网络要求】

  - 首次部署：需要联网下载前端库、地图瓦片和 GeoJSON 数据
  - 日常运行：完全脱网，无需互联网连接
